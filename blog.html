1. What is DOM and what is the purpose of DOM?

Answer : The full form of dom is "Document Object Element". DOM mainly converts an HTML file or document into an object. It is the most important feature of javascript. It is an interface for a web page. Javascript can control Html by using dom. Using dom javascript can change many things dynamically on the web page and it also can change tags, elements, and attributes, and remove or delete HTML and CSS elements, attributes, styles, etc. Using Dom Event "onclick", "addeventlistener" it can make a button clickable. Without these Dom has many using purposes.

3.What is event bubble?

Answer :Event bubbles are a concept in the DOM (Document Object Model).An event bubble is a type of DOM event where the event is first clicked on the innermost element, then successively clicked on the element preceding the target element until it reaches the outermost DOM element or document object.

4.What is a callback function and why will you use it?

Answer :Definition of callback function It is a function that is created after another function is created it is called callback function. While coding we have to code a lot of javascript but if the same task is done in 2 or more places, instead of doing one task repeatedly, after creating a function, if we write the name and parameters of that function in another function i.e. call, then There will be no need to do the same. No need to write again and again. Thus we can simplify and shorten our work by using callback functions.
